SVG Converter

Plugin Identifier: 20000

Summary

This plugin is a 4D implementation of the rsvg program.

You can convert SVG to PDF, PNG, PS or SVG.

Convert Many

SVG Convert array

error:=SVG Convert array (images; data; type; keys; values; backgroundColor; baseUri)

images (ARRAY)

Pass an array of SVG images.

data (BLOB)

On return, a single document of the images concatenated.

type (LONGINT)

Optional. Target type. Must be either PDF or PS. Default is PDF.

You can use a SVG Output Formats constant for type.

keys (ARRAY)

Optional. An array of longint key names to configure.

You can use a SVG Converter Options constant for keys.

values (ARRAY)

Optional. An array of real key values to set.

backgroundColor (TEXT)

Optional. Background color in valid CSS notation. black, white, #abccee, #aaa...

baseUri (TEXT)

Optional. Base URI of the SVG.

error (LONGINT)

On success, 0. -1: svg parse error, -2: image is not svg, -3: invalid export format .

SVG Convert

error:=SVG Convert (image; data; type; keys; values; backgroundColor; baseUri)

image (PICTURE)

Pass an SVG image.

data (BLOB)

On return, the image converted.

type (LONGINT)

Optional. Target type. PNG, SVG, PDF or PS. Default is PDF.

You can use a SVG Output Formats constant for type.

keys (ARRAY)

Optional. An array of longint key names to configure.

You can use a SVG Converter Options constant for keys.

values (ARRAY)

Optional. An array of real key values to set.

backgroundColor (TEXT)

Optional. Background color in valid CSS notation. black, white, #abccee, #aaa...

baseUri (TEXT)

Optional. Base URI of the SVG.

error (LONGINT)

On success, 0. -1: svg parse error, -2: image is not svg, -3: invalid export format .
